
# Projeto: Mentoria de Programação e Fundamentos de Computação com IA

## 1. Contexto e Objetivos
Este caderno temático foi criado para apoiar o estudo intensivo de **Ciência da Computação**, com foco no primeiro semestre da graduação. O assunto de interesse abrange desde o **alicerce matemático** (Pré-Cálculo e Teoria dos Números) até as **práticas profissionais de engenharia de software** (Clean Code, Algoritmos e Controle de Versão).

**Objetivos de Estudo:**
*   Dominar a lógica de programação começando por HTML, CSS e JavaScript.
*   Aplicar princípios de **Código Limpo** para escrever software legível e profissional.
*   Entender a eficiência algorítmica através da **Notação Big O**.
*   Utilizar o **Git e GitHub** como "máquina do tempo" e ferramenta de integração desde o primeiro dia.
*   Superar as barreiras de aprendizagem (neurodiversidade) através de explicações simplificadas e diretas.

## 2. Curadoria de Fontes
As fontes selecionadas e analisadas para este projeto são:
1.  **Código Limpo (Robert C. Martin):** Foco em funções pequenas, nomes significativos e qualidade de código.
2.  **Entendendo Algoritmos (Aditya Bhargava):** Guia ilustrado sobre estruturas de dados e eficiência.
3.  **Controlando Versões com Git e GitHub (Casa do Código):** Guia prático de versionamento e trabalho em equipe.
4.  **Pré-Cálculo (Marcos Lima):** Fundamentos matemáticos necessários para computação.
5.  **A Regra é Não Ter Regras (Reed Hastings & Erin Meyer):** Cultura de alta performance e densidade de talento.
6.  **Dev em Dobro (YouTube Transcript):** Planejamento e armadilhas para programadores iniciantes.

## 3. Engenharia de Prompts 
Durante a interação com o NotebookLM, as seguintes estratégias foram testadas:

  *Prompt de Diagnóstico:* *"chat, com tudo o que te dei, sabe sobre programação em qual nível?"*
    *   **Resposta:** A IA mapeou o conhecimento desde o nível iniciante até arquitetura avançada, podendo ajudar a pessoa em qualquer jornada
   **Prompt de Personalização:** *"chat, eu preciso da sua ajuda pra aprender isso, não consigo ver video, não consigo ler livro, é eu e tu, tu e eu..."*
    *   **Dificuldade:** No início, a IA sugeria assistir vídeos. Foi necessário reforçar a condição de **autismo** e a necessidade de texto "mastigado" e direto.
    *   **Troubleshooting:** O chat identificou que não acessa links externos (Livraria Uninter), exigindo o upload direto dos arquivos para processamento local.
   **Prompt Matemático:** *"a e bhaskara?"*
    *   **Resultado:** A IA simplificou o papel do coeficiente "a" na fórmula quadrática, conectando-o à prática de programação e ao formato padrão $ax^2 + bx + c = 0$.

## 4. Miniguia de Estudo 

### Resumos Estruturados
*   **Matemática Base:** Números racionais são frações de inteiros $\frac{a}{b}$ ($b \neq 0$). Em computação, são representados por *floats*, onde erros de precisão podem ocorrer.
*   **Git Essencial:** O comando `git init` cria um repositório local. O fluxo básico consiste em `git add` (rastrear), `git commit` (gravar) e `git push` (compartilhar).
*   **Clean Code:** O código deve ser lido como uma narrativa, de cima para baixo (Regra Decrescente). Se uma função precisa de comentários para ser entendida, ela provavelmente deve ser simplificada.

### Glossário de Conceitos
*   **Big O:** Medida de eficiência que descreve como o tempo de execução cresce conforme o número de itens aumenta.
*   **Recursão:** Quando uma função chama a si mesma, exigindo sempre um caso-base e um caso recursivo para evitar loops infinitos.
*   **Princípio do Astro de Rock:** Ideia de que um engenheiro excepcional é muito mais valioso que vários medianos devido à sua criatividade e visão de padrões.

### Prompts Reutilizáveis para Revisão
*   *"Explique o conceito de [Tópico] da Aula [Número] focando na aplicação prática dentro de um script JavaScript."*
*   *"Como posso refatorar esta expressão algébrica seguindo os princípios de Nomes Significativos do Clean Code?"*
*   *"Dada a equação [Equação], identifique os coeficientes a, b e c e resolva o discriminante passo a passo."*
